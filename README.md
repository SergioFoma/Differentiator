# Differentiator 📝

## Описание ✍️

```Differentiator``` - это учебный проект, написанный для изучения алгоритмов работы с деревьями. Программа считывает из файла математическое выражение,
в рекурсивном спуске строит по нему дерево и результаты дифференцирования пошагово записывает в LaTeX - файл

# Установка программы 👾

### **Чтобы скачать весь проект, нужно:**

1. В Вашем терминале необходимо указать директорию, куда вы хотите скачать данный проект.

2. Затем в терминале введите:
```
git clone git@github.com:SergioFoma/Differentiator.git
```

3. Дождитесь скачивания.

4. После чего будет создана рабочая директория с названием репозитория.

# Запуск программы 🚀

1. Чтобы скомпилировать программу в консоли в корневой папке необходимо ввести команду:
```
make
```

2. Для запуска дифференциатора необходимо ввести в консоли команду:
```
Build/Result.out
```
# Возможности 📐

- Парсинг математических выражений
- Символьное дифференцирование по переменной
- Вывод результата в LaTeX-формате
- Генерация PDF c подробным пошаговым выводом

# Использование 📚📒

1. Создайте текстовый файл и запишите в него математическое выражение. Пример:
```
В файле mathExpression.txt:

exp(10)*ln(17)*sin(x)$

```

>⚠️ Математическое выражение должно оканчиваться ключевым символом ```$```

2. Запустите скомпилированную программу:

```
Build/Result.out
```

3. Дифференциатор сделает два запроса:

    - Файл, куда Вы хотите сохранить код LaTeX
    - Файл, в котором дифференциатору необходимо искать математическое выражение

Пример:
```
Enter the name of file, where i will save latex dump: expressionLatex.tex
Enter the name of file, where i will find mathematical statement: mathExpression.txt
```

4. При успешном завершении дифференцирования программа создаст LaTeX и pdf файлы с пошаговым
дифференцированием.

```tex
\[ \frac{ d }{ dx }( sin  \left(  x  \right) ) = cos  \left(  x  \right) \cdot  1  \]
\[ \frac{ d }{ dx }( ln \left(  17  \right) ) = \frac{ 0 }{ 17 } \]
...
```
